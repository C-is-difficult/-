📐 横向自由振动动力学方程

根据牛顿第二定律，弹性棒无阻尼自由横振动满足：

∂⁴y/∂x⁴ + (ρS/EJ)·∂²y/∂t² = 0

采用分离变量法 y(x,t)=X(x)·T(t) 求解，得到两组独立常微分方程。

🔗 边界条件与本征值

自由悬挂边界：弯矩、横向作用力均为0，最终特征方程：

cos(KL)·cosh(KL) = 1

数值求解得到本征值：KL = 4.730, 7.853, 10.996, 14.137,...

奇数阶为对称振动，偶数阶为反对称振动。

📈 倍频法完整公式推导

1.角频率：ωₙ = Kₙ²·√(EJ/ρS)

2.代入βₙ=KₙL：ωₙ = βₙ²/L²·√(EJ/ρS)

3.频率转换：fₙ = βₙ²/(2πL²)·√(EJ/ρS)

4.线性形式：fₙ = k·βₙ²（全局线性拟合）
